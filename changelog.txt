Tikiwiki Changelog
------------------

Changelog is now generated by cvs2cl.pl
http://cvsbook.red-bean.com/cvsbook.html#cvs2cl%20--%20Generate%20GNU-Style%20ChangeLogs
This script lists chronologically the comments made with cvs commits.
For example, for 1.9.7 : cvs2cl -F BRANCH-1-9 --delta REL-1-9-6:BRANCH-1-9
produces a file named Changelog, then in vim :%s/\n\(\s*\*\)/\1/

Use the following tags to distinguish changes :
* [NEW] apply if something new was added in tikiwiki
* [MOD] is an enhancement; not really new but makes things work better
* [FIX] can be used for bug fixes of any sort
* [SEC] for security fix operations
* [REL] changes for release process only
* [KIL] removal of unused or obsolete files
* -[DB] for changes in the database
when possible, it's also nice to indicate what feature is concerned by 
the change.
